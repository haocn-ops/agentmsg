# AgentMsg Admin Dashboard

Simple admin dashboard for AgentMsg platform.

## Quick Start

```bash
cd sdk/javascript/admin-dashboard
npx serve .
```

Then open http://localhost:3000 in your browser.

## Features

- Login with API key
- View all registered agents
- Real-time message statistics
- Agent status monitoring
- Message log view

## Usage

1. Enter your API key to login
2. View dashboard with agent list and statistics
3. Monitor agent status and capabilities
